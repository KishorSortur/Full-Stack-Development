What are localStorage and sessionStorage?

localStorage and sessionStorage are part of the Web Storage API in HTML5. They allow web pages to store key-value pairs in a user's browser.

localStorage saves data with no expiration time — it remains even after the browser is closed or the computer is restarted.

sessionStorage, on the other hand, stores data only for the duration of the page session — meaning the data is lost when the tab or browser is closed.

These are commonly used to save user preferences, temporary form data, or authentication states on the client side.
